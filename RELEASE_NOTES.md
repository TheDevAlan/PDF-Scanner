# Release Notes - Version 1.3.0

## PDF Scanner with PDFium

This version brings important changes and improvements:

### 🚀 Highlights

- **New PDF Engine**: Switched from pdf2image/Poppler to the faster and more license-friendly pypdfium2 library
- **Improved User Interface**: Sharper display on high-DPI screens, modern fonts, and better scaling

### 🛠️ Installation Instructions

1. Download the complete package
2. Extract the ZIP file
3. Run `PDF_Scanner.exe` directly - no separate installation necessary

### 📝 Important Notes

- The new version no longer requires external installation of Poppler
- All libraries are included directly in the application
- The application is now DPI-aware and scales correctly on high-DPI screens
